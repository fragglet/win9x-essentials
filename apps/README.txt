
7z920.exe      - 7-zip 9.20. Fully-featured decompression tool that (unlike
                 others like WinZip or WinRAR) does not have shareware nag
                 screens. Supports most archive formats.

daemn347.exe   - Daemon Tools 3.47. Virtual CD/DVD drive that can mount .iso
                 images as though they're on a real drive. Useful for playing
                 games that require a CD to be present, without needing an
                 actual drive or disk.

dlmanip        - Drive letter manipulator. Useful to fix a freezing issue
                 when a floppy drive is not present.

endital2.exe   - EndItAll 2.0. More fully-featured replacement for Task
                 Manager.

fzla2215.exe   - FileZilla 2.2.15. FTP client.

krnex452.exe   - KernelEx 4.5.2. Allows some software designed for Windows
                 NT to be run on 9x.

myslow.b13     - MySlow beta 13. Slows down DOS programs that run too fast.

ps2rate.04     - Adjusts sampling rate for PS/2 mouse.

putty063.exe   - PuTTY 0.63 SSH client.

rawwrite.07    - Utility for writing raw floppy disks.

vdmsound.a3    - VDMSound alpha3 version for Windows 9x. Emulator of legacy
