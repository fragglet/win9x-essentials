
bm437.fon    - Replacement "Terminal" font using the original IBM font
               characters, from the "Oldschool PC Fonts" pack (int10h.org).
               This gives the most accurate possible display in DOS windows.

core-fonts   - Microsoft Core fonts for the Web, standard set of common
               TrueType fonts that most people are familiar with nowadays.
               These are included with Windows 98 but not 95.

ttfext.exe   - Microsoft Font properties extension, version 2.1. Adds a
               properties pane to Explorer for .ttf files.
